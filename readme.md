## Requirements
### python virtual environment
```sh
pip install virtualenv
python -m venv env
source env/bin/activate
```

### Install DBT core
```sh
python -m pip install dbt-core
```

### Install DBT for DuckDB
```sh
python -m pip install  dbt-duckdb
```

### Install DuckDB
```sh
brew install duckdb
```
